This directory contains Request for Comments (RFCs) for the Busy Office project.

## Reading Order

1. **RFC-001: Game Architecture** - Vision, two modes, tech stack decisions
2. **RFC-002: ECS Integration** - unify-ecs + Arch backend, component/system design
3. **RFC-004: C#-GDScript Bridge** - GameBridge autoload (needed to understand data flow)
4. **RFC-003: Office Simulation** - Tile grid, pathfinding, agent FSM
5. **RFC-005: A2A WebSocket** - Connected mode, tentacle-punch integration
6. **RFC-006: Tycoon Mode** - Offline mode, task generation, economy

## RFC Process

- **Draft**: Initial proposal, open for changes
- **Accepted**: Approved for implementation
- **Implemented**: Fully implemented
- **Superseded**: Replaced by another RFC
